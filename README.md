DOGE-to-USD
===========
Simple DOGE to USD converter 
--------------

**USAGE:** 
> ruby balance.rb [DOGE BALANCE] [INITIAL USD INVESTMENT]

**EXAMPLE:** 
> ruby balance.rb 118000 95
